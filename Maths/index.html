Conway's Game

Conway's game of life is a set of rules for cellular automata. Cellular automata is a system in which there is a grid of cells that are either fillled or empty depending on the states of the surrounding cells. In Conway's game of life, a 'live' cell dies if there are fewer than two other live cells surrounding it. We call this under population. A live cell dies if there are greater than three other live cells surrounding it. We call this overpopulation. A live cell stays alive if there are two or three other live cells surrounding it. If a 'dead' cell has exactly three surrounding live cells, it is born into a live cell
